# tym2muv - Product Documentation

## Overview
**tym2muv** is a premium real-time real estate marketplace designed for the African market. It connects property seekers with verified agents and individual sellers through a high-performance, mobile-first web application.

## Core Features

### 1. Advanced Property Search & Discovery
*   **Location-Based Search:** Filter properties by country (Ghana, Nigeria, Kenya, etc.) and specific regions/cities.
*   **Smart Filters:** Narrow down results by property type (Apartment, House, Land, etc.), price range, number of bedrooms/bathrooms, and listing type (Rent vs. Sale).
*   **Dynamic Categories:** Specialized categories for Residential, Commercial, Land, and Short-let properties.
*   **Real-Time Sorting:** Listings are sorted by relevance, premium status, and date posted.

### 2. Premium Listing System
*   **Priority Ranking:** Premium listings are boosted to the top of search results for the first 10 days.
*   **Extended Visibility:** Premium ads stay live for 91 days, while standard ads stay for 35 days.
*   **Visual Distinction:** Premium listings feature a unique "Award" badge and enhanced card styling to attract more views.

### 3. Real-Time Communication
*   **In-App Chat:** Secure, real-time messaging between buyers and sellers.
*   **WhatsApp Integration:** One-click access to chat with agents directly on WhatsApp.
*   **Direct Calling:** Integrated click-to-call functionality for quick inquiries.

### 4. Multi-Currency & Localization
*   **Automatic Conversion:** Prices are automatically converted based on the user's selected country (GHS, NGN, KES, USD).
*   **Localized Content:** Featured properties and search defaults adapt to the user's region.

### 5. Seller & Agent Tools
*   **Property Management:** Owners can post, edit, and manage their listings through a dedicated profile.
*   **Verification System:** Agents can earn a "Verified" badge to build trust with potential clients.
*   **Rich Media Support:** Support for multiple high-resolution photos, videos, and virtual tour links (e.g., Matterport).

### 6. User Experience & Design
*   **3D Interactive Cards:** Engaging property cards with 3D flip effects and image carousels.
*   **Glassmorphism UI:** Modern, clean aesthetic using glass-style overlays and smooth animations.
*   **Mobile-First Navigation:** Optimized bottom navigation bar for seamless one-handed use on mobile devices.

## Technical Stack
*   **Frontend:** React 18, TypeScript, Vite.
*   **Styling:** Tailwind CSS (Utility-first).
*   **Animations:** Framer Motion.
*   **Icons:** Lucide React.
*   **State Management:** React Context API (Auth, Location).
